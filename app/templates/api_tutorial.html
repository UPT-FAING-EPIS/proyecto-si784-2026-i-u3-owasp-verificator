La API y los esquemas necesarios están disponibles en producción, listos para integrarse como funciones o plugins.
Esquema de Funciones Disponibles
skill_analyze_url POST
Analiza una URL buscando cabeceras HTTP faltantes (CSP, HSTS, CORS) e información sensible expuesta.
POST /analyze/api
skill_analyze_code POST
Realiza un escaneo estático de código buscando SQLi, XSS, secretos hardcodeados y funciones prohibidas.
POST /analyze/api
skill_analyze_github POST
Descarga y audita un repositorio GitHub. Soporta repos privados utilizando el token guardado en cuenta.
POST /analyze/api
skill_get_report GET
Recupera el historial y listado de hallazgos para un reporte de escaneo específico mediante su ID.
GET /reports/api/{id}

Claves de Acceso a la API

Genera un token API único para autenticar tus herramientas externas (CLI, Scripts, Webhooks) enviándolo en la cabecera X-API-Key en cada petición HTTP.

Referencia de Endpoints

Método Endpoint Descripción
POST /analyze/api Ejecuta un análisis de código o URL activa.
GET /reports/api/{id} Obtiene detalles de los hallazgos para un escaneo.
GET /health Valida el estado del servicio y estadísticas generales.

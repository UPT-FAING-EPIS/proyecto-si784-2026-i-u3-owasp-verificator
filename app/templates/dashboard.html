Módulos de Auditoría OWASP

OWASP Verificator ejecuta auditorías de seguridad automáticas sobre el código fuente local, URLs de producción y repositorios de Git para identificar riesgos del Top 10.

Detección de Secretos

Localiza contraseñas, tokens privados, certificados y claves de API expuestas en el código fuente.

Funciones Peligrosas

Identifica el uso de llamadas dinámicas inseguras como eval() o ejecución de subprocesos no saneados.

Saneamiento de Entradas

Inspecciona flujos de datos expuestos a inyecciones de código, SQL Injection y Cross-Site Scripting (XSS).

Seguridad en Contenedores

Audita Dockerfiles para validar el principio de menor privilegio, puertos no seguros y paquetes desactualizados.

Para comenzar: Realiza tu primer escaneo estático o dinámico haciendo clic en Nuevo análisis en la barra superior.
